📌Основні команди навігації в Linux (bash / термінал):
Команда	             Що робить
cd папка	        Перейти в підпапку
cd ..	            Повернутись на рівень вище
cd ../..	        Повернутись на два рівні вгору
cd	                Перейти в домашню директорію
cd /повний/шлях	    Перейти за абсолютним шляхом

Якщо в назві є пробіли:
cd "TASK 2"      або          cd TASK\ 2

Навигация:
pwd                   «Где я?» (показывает полный путь к текущей папке).
ls -F                 показать файлы в текущей папке (флаг -F пометит папки косой чертой /).
ls -la                показать файлы в текущей папке       -l — (long format) показывает подробный список: права доступа, владелец, размер, дату и т.п.
                                                           -a — (all) показывает все файлы, включая скрытые (те, что начинаются с .).

cd projects/backend   перейти в папку.
cd ..                 подняться на один уровень выше.
cd ~                  быстро вернуться в домашнюю папку пользователя.
dir                   Показати зміст активної директорії (які файли і папки знаходяться в цій адоесі)
mk                    Создать (dir - директорию)

Работа с файлами:
mkdir my_folder                 создать новую папку.
touch server.ts                 создать пустой файл.
echo "Привет из локального облака!" > hello.txt   создаст файл hello.txt и добавит туда этот текст
cat <имя файла>                 открывает файл для чтения
cp file.txt new_file.txt        скопировать файл.
mv file.txt /another/folder/    переместить файл.
rm file.txt                     удалить файл (осторожно, в Linux нет «Корзины», удаляется сразу!).
cat file.json                   быстро вывести содержимое файла на экран.
rmdir название_папки            удалить директорию
rm -rf название_папки           -r (recursive) — удаляет папку и всё её содержимое (все вложенные файлы и папки).
                                -f (force) — «насильно», не переспрашивая тебя о подтверждении для каждого файла.

Системные:
clear                       очистить экран терминала (чтобы не путаться).
history                     показать список последних команд, которые ты вводил.
sudo ...                    выполнить команду от имени администратора (потребует твой пароль).
top или htop                диспетчер задач (посмотреть, не завис ли твой Node.js и сколько памяти он ест).

/mnt                        это папка, куда монтируются диски, например твои Windows-диски C, D и т.д.
sudo                        указывает на то что команда идёт от админа
apt                         аналог Microsoft Store только в Debian
nvm                         Node Version Manager используеться в Debian, аналог обычного npm
Get-Command имя_программы   прказывает директориб коннкретно установленой программы
ping                        это сетевая утилита для проверки доступности узлов в IP-сетях, в Linux лучше её использовать вмести с командой sud, чтобы подчеркнуть свои права доступа (администратора)
explorer.exe .              открывает обычный проводник, точка указывает но то что: "открой мне пожалуйста проводник в этой директории"



Как привязать VS Code из Windows в Debian:
        - Открой VS Code просто из Windows.
        - Нажми Ctrl + Shift + P (откроется командная палета).
        - Введи: "WSL: Connect to WSL".
        - VS Code подключится к твоему Debian, и ты сможешь через File -> Open Folder выбрать свою папку проекта (путь будет выглядеть как \\wsl$\Debian\home\oleksii\...).


Если мы используем терминал в Линуксе (Debian), а проект наш находиться на Windows, не беда мы можкм командой:
        cd /mnt/d <путь к проекту> перейти в наш проект


➡️ Как узнать какие версии Node.js достыпны мне в моей ОС:
nvm ls-remote или просто без флага nvm ls - покажет все версии Node.js, которые можно установить
which node - быстрый способ понять, что у тебя используется и где установлено
        При создании проектов разроботчики вносят данные о версии Node.js в отдельный файл .nvmrc
                nvm use - для быстроко переключения версии Node.js которая указана в .nvmrc
nvm install 24 - обновляем версию Node.js, если ксть в этом необходимость
nvm alias default 24 - устанавливаем версию 24 по умолчанию
