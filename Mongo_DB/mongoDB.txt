Початок роботи та налаштування:
https://cloud.mongodb.com/ - региструємося, в верхньому лівому куту вілкриважмо список баз даних и створюємо новий проект в якому будуть розміщуватися наші бази даних:
- Create a Project;
- в поле вводим ім'я - next;
- якщо треба то добавляжмо Add Members and Set Permissions - Create Project.

Після того як ми створили проект його треба налаштувати:
- Security - Database Access - створюємо логін та пароль завдяки якому ми будемо підключатися до нашої БД;
- Add New Database User - в поле username вводимо логін, який бкде підкобчатися до нашої БД цього проекту - в поле password вводим пароль або генеруємо автоматично;
- після цього записуэмо логінта пароль;
- Add Built in Role - вибираэмо роль наприклад Atlas admin;
- Add User;

- Потрібно вказати, з якої IP адреси ми можемо підключитися до проекту - Network Access;
- Add IP Address - в поле вводимо адресу ADD CURENT IP ADDRESS або на етапі розробки тим пачи коли працює команда вибираємо ALLOW ACCESS FROM ANYWHERE.


Створюємо в MongoDB Cluster (кластер):
- Built a Cluster - і обираємо, який тарифний план нам підходить, потім Regio - це де буде знаходитися сервер (бажано по ближче від місця де ти знаходишся) - Create Deployment

- Browse Collections - тут відображаються наші бази даних. Для створення бази нажимаємо Add My Own Data - вводимо назву бази даних і назву таблиці - Create. (Load a Sample Dataset - це завантажує приклади бази даних)

Для зрусності праці з MongoDB потрібно встановисти на ПК MongoDB Compass https://www.mongodb.com/products/tools/compass
- для підключеня її до MongoDB, нажимаємо +Add New Conection - і в строку url вставляємо посилання з MongoDB в тій БД яку ми хочемо підклюсчити. В обраній БД вибираэмо conect - compass і копіруємо строку підключення і вставляємо її в compass, тільки не забуваємо записати пароль.
- завдяки compass ми можемо не вручну додавати json а імпортувати цілий файл.


Видалення проект:
- для того щоб видалити проект, спочатку треба очистити всі кластери в проекті;
- переходимо в проект - кластери - в строці біля назви кластеру вибираєио три крапочки и Terminate - вводим назву кластера і підьрерджуємо;
- переходимо в View All Projects - і успішно видяляжмо проект.
